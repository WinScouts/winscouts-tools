SMS Profile URL Extractor

One-shot tool — paste your SMS member list, download a membership number → profile URL CSV

How to use this tool

  1. In the Scout Membership System, open the member list for Winchester District
  2. Select all rows (Ctrl+A or Cmd+A) and copy (Ctrl+C or Cmd+C)
  3. Click in the paste area below and paste (Ctrl+V or Cmd+V)
  4. Click Extract URLs — the tool will find all profile links
  5. Download the CSV and share it for use in the Accreditations dashboard
